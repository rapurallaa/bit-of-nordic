# Images

Upload your item photos here. One folder per category.

## Folder structure
```
images/
  watches/      → e.g. omega-seamaster-1965.jpg
  coins/        → e.g. finnish-markka-1917.jpg
  glass/        → e.g. iittala-vase-1960s.jpg
  jewellery/    → e.g. silver-brooch-1940s.jpg
```

## Photo tips
- Shoot on a neutral background (white, linen, or slate grey work well)
- Good natural light or a lightbox
- Landscape or portrait both fine — the site handles both
- JPG or PNG, ideally 1200px or wider on the longest side
- Name the file to match the item `id` in `items.json`

## How to upload via GitHub
1. Go to the folder (e.g. `public/images/watches/`)
2. Click **Add file → Upload files**
3. Drag and drop your photos
4. Click **Commit changes**
5. Vercel auto-deploys within ~30 seconds
